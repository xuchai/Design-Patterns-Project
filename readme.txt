
This is a project using different design patterns. Codes are in “src” folder and the UML models are in “Models” folder. Please check it.

The detailed project description is listed below:

Coding/programming Environment:
     	Eclipse Mars IDE;
     	Github version control;

Design Patterns Used:
     	Builder
     	Observer
     	Factory
     	Bridge
     	Momento
     	Iterator
     	Strategy
     	Facade
     	Chain of Responsibility

Project Senario:   
	A new zoo:
	People (Staff + Visitors);
	Animals (Seal, Rabbit, …);
	Observers;
	Staff have different roles (cleaning, training and feeding);
	Animals have different sounds (like bark);
	Ticket office;
